Tag Library

Every tag used on emails, forms, and cadences. The registry lives in data/tag-library.json in this repo — the validator checks every manifest tag against it, and the Campaign Builder resolves tags from it. To add or change a tag, edit that file (never invent tags ad hoc).

Registered Campaign Tags

Campaign tags are minted per campaign (format campaign-[name]-[q#-yyyy]) and registered here once in use.
